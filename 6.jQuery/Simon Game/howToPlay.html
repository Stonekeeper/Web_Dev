How to Play?

1. First the computer flashes the color to be Clicked.

2. The player must remember the flashed color and click the corresponding color.

Once the user clicks the corresponding color, The Computer flashes the second color.

3. Now the user must click first flashed color and then the second flashed color.

4. Now the computer will flash the third color, Now the user must select first, second and the third flashed colors.

So on and so forth

6. If the user clicks the wrong sequence the game is over.
